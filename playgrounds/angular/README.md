# Angular Playground

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn start
```

### Compiles and minifies for production
```
yarn build
```

## Running end-to-end tests

```
yarn e2e
```

Run `yarn e2e` to execute the end-to-end tests via [Protractor](http://www.protractortest.org/).
